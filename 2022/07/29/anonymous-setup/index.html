Fully Anonymize Kali With Whonix

Setup

Firstly you will need to install VirtualBox in order to run Whonix

You can install it directly from the official website https://www.virtualbox.org/
After downloading it we will need to install our Whoonix Gateway and Workstation
Basically we want to route all our Kali traffic through Whonix in order to stay anonymous.

We will install the windows host version because on our host we are running Windows
Once the Whonix-XCFE is installed we will need to import it on VirtualBox

You need to make sure that the Gateway have those settings:

Once you finish veryfying you can start the Gateway
You will be prompted with security questions, just answer yes to all
After finishing all the questions you will be asked to connect to Tor

Press Next, connect to Tor and change your password

Do the same thing on the Workstation too
Your final configuration should look something like this:

Once you have everything like this we will need to make a couple more settings and we will be done
Open Kali’s network settings and set it to whonix internal network

Boot the machine and enter this configuration: nano /etc/network/interface

iface eth0 inet static
address 10.152.152.11
netmask 255.255.192.0
gateway 10.152.152.10

Now we will need to edit this configuration: nano /etc/resolv.conf

nameserver 10.152.152.10

We will need to apply the changes by restarting the interface.
In order to do that type:

sudo ifdown eth0
sudo ifup eth0
