    EXPLOITATION

    As we can see we have access to /nav.php but when we click to a link we get redirected back to /index.php, so we need to find a way to bypass that

    So in order to get rid of that we can use BURPSUITE by intercepting the request -> Do intercept -> Response to this request and keep forwarding until you can see the HTTP STATUS CODE 302 and change it to 200 OK

    Modify it and then forward it again.

    As you can see we successfully bypassed the restriction.
    Now we can create a BLANK account:

    admin:admin

    And try to access it via the login form from /login.php
